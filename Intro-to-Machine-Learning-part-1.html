Building Your Model

he steps to building and using a model are:

Define: What type of model will it be? A decision tree? Some other type of model? Some other parameters of the model type are specified too. Fit: Capture patterns from provided data. This is the heart of modeling. Predict: Just what it sounds like Evaluate: Determine how accurate the models predictions are.

Model Validation

Mean Absolute Error (also called MAE) error=actualpredicted

With the MAE metric, we take the absolute value of each error. This converts each error to a positive number. We then take the average of those absolute errors. This is our measure of model quality. In plain English, it can be said as

On average, our predictions are off by about X.

https://www.kaggle.com/dansbecker/model-validation
